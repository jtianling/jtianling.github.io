## Local development

- Ruby `3.3.x`
- Bundler `2.5+`
- Install: `bundle install`
- Serve locally: `bundle exec jekyll serve --host 127.0.0.1 --port 8080`

Set `google_analytics` in `_config.yml` to a GA4 measurement ID such as `G-XXXXXXXXXX` if you want analytics enabled.
